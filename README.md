## Дипломный проект. 
## Задание 1: Юнит-тесты

## Описание проекта
В проекте написаны юнит-тесты для проверки программы, которая помогает заказать бургер в Stellar Burgers

### Реализованные сценарии

Созданы юнит-тесты, покрывающие классы `Bun`, `Burger`, `Ingredient`, `Database`

Процент покрытия 85% (отчет: `htmlcov/index.html`)


### Структура проекта

- `praktikum` - пакет, содержащий код программы
- `tests` - пакет, содержащий юнит-тесты
- `.coverage` - файл для отчёта о % покрытия
- `conftest.py` - Фикстуры
- `README.md` - Описание проекта

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск автотестов и создание HTML-отчета о покрытии**

>  `$ pytest --cov=praktikum --cov-report=html`
